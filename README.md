Resumai 🚀
The Ultimate AI-Powered, ATS-Optimized Resume Builder.

Resumai is a modern, privacy-focused web application designed to bridge the gap between your professional experience and the job you want. Using Google’s Gemini AI, it proactively tailors every section of your resume—from professional summaries to achievement bullet points—against any job description you provide.

✨ Key Features
🤖 Proactive AI Tailoring: Resumai doesn't just format your data—it intelligently rewrites and expands your experience to match specific Job Descriptions (JD).
📋 Structured Input System: Clean, row-based forms for Experience, Education, Projects, Test Scores, and more.
📄 LaTeX-to-PDF Engine: Generates high-fidelity, ATS-friendly PDF documents.
📂 Local Archive (IndexedDB): Securely saves your generated resumes in your browser's database. No cloud storage needed.
📲 Native Sharing: Integrated Web Share API to instantly send resumes via WhatsApp, Email, or Slack.
💎 Premium UI: A sleek, mobile-first glassmorphism design.
🔒 Privacy First: Your Gemini API Key is stored only in your local browser storage.
🚀 How to Use
💻 On PC (Windows/Mac/Linux)
Clone/Download: Clone this repository or download the ZIP file and extract it.
Launch: Simply double-click index.html to open it in your favorite browser (Chrome, Edge, or Brave recommended).
Setup: Go to the Settings tab and paste your Google Gemini API Key.
Generate: Create a profile, select it, paste a Job Description, and hit Generate.
📱 On Mobile (Android/iOS)
Since this is a web app, you can use it on your phone easily:

Hosting: Upload this folder to GitHub Pages, Netlify, or Vercel (all are free).
Access: Open your unique hosting URL in your mobile browser.
App Experience: For the best experience, tap your browser's "Share" or "Menu" button and select "Add to Home Screen". It will now work and feel just like a native app.
Share Instantly: After generating a resume, use the "Share" button to instantly send the PDF via WhatsApp or Email without needing to download it first.
🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript (ES6+)
AI: Google Gemini API
Storage: IndexedDB & LocalStorage
PDF Compiler: LaTeX-on-HTTP
